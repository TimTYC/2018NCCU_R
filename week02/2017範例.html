Ch02: 條件判斷語句 - if

條件判斷1

  • 一般而言，R語言的程式碼是由上自下依序執行
  • 有時需要根據某些條件判斷執行某些分支
  • 在R中使用if來進行判斷
  • 將敘述內容寫在大括號{ }裡面
# 判斷x是否大於2, 如果是，則印出結果
# cat是什麼？輸入：?cat
x <- 4
if(x > 2){
  cat(x, "is larger than 2")
}

條件判斷2

  • 在R中使用if來進行判斷
  • 將敘述內容寫在大括號{ }裡面
  • 當敘述句只有一行時，可以省略大括號
# 判斷x是否大於2, 如果是，則印出結果
x <- 4
if(x > 2) cat(x, "is larger than 2")

條件判斷3

  • 若需要兩個判斷分支時，加入else
y <- 1
if(y > 2){
  cat(y, "is larger than 2")
}else{
  cat(y, "is smaller than 2")
}

條件判斷4

  • 若需要兩個判斷分支時，加入else
  • 可以使用ifelse函數進行簡化，而且該函數支援向量化運算 (後述)
y <- 1
ifelse(y > 2, "larger than 2", "smaller than 2")
z <- 1:5
ifelse(z > 2, "larger than 2", "smaller than 2")

條件判斷5

  • 若需要多個判斷分支時，加入if, else if, ..., else
y <- 2
if(y > 2){
  cat(y, "is larger than 2")
}else if(y < 2){
  cat(y, "is smaller than 2")
}else{
  cat(y, "is equal to 2")
}

雙重循環語句

  • 可用來重複執行某段程式碼

k <- 0
for(i in 1:3){

  }
  cat("\n")
}

Ch04: 自訂函數 - function

自訂函數 (function)

  • 函數是R語言的基石，有利重複使用
  • 通過多個函數的組裝，可以完成更複雜的任務
  • 函數在子環境中執行，不對外產生影響
  • 當有需要經常做這件事的時候，可以考慮寫成一個函式
myfun <- function(x){
  # 函式內容
}

自訂函數

  • 條件判斷
myif <- function(x){ # 這個function有一個變數x
   if(x > 2){
    cat(x, "is larger than 2")
  }else if(x < 2){
    cat(x, "is smaller than 2")
  }else{
    cat(x, "is equal to 2")
  }  
}
myif(1); myif(2); myif(3)

自訂函數

  • 循環語句
  • 我發明了一個運算法則：一個數字x:$\sqrt{(x*\pi)+1+2+3+4+5}$，
myfun <- function(x){ # 這個function有一個變數x
  x <- pi*x
  for (i in 1:5) {

myRead <- function(fname, path0="data/movies/"){
  mov <- list()
  for(i in 1:length(fname)){
    path <- paste(path0, fname[i], sep="")
    mov[[i]] <- read.csv(path)
  }
  do.call(rbind, mov)
}

Ch05: Pipe Line Coding Style

2014 年起最有影響的套件之一：magrittr

  • 壓縮的程式碼不好讀
  • 展開的程式碼會產生很多暫存變數
  • 套件magrittr部份解決了這個問題
  • 基本算子 %>%, %<>%, %T>%, %$%（詳細說明）
  • Pipeline快捷鍵：ctrl / command + shift + M

基本算子 (%>%)

  • 想像一下程式的寫作與閱讀邏輯

基本算子 (%>%)

  • %>% 會將算子左邊的物件 (object) 傳到右邊的函數 (function) 中第一個argument
  • . 點號適合用在欲傳入變數不是在傳入函數的第一個位置時使用
  • use x %>% f, rather than f(x)
  • or use x %>% f(y, z), rather than f(x, y, z)
  • or y %>% f(x, ., z), rather than f(x, y, z)
# install.packages("magrittr")
library(magrittr)
x <- 1:10
mean(x)
x %>% mean # 由左而右順序操作

幾種等價用法

利用三角形面積公式說明%>%算子的幾種等價用法

tri_area <- function(a, h=5) a*h/2 # 大括號省略
a <- 10
tri_area(a)
a %>% tri_area          # 省略括號
a %>% tri_area(h=5)     # 保留括號
a %>% tri_area(., h=5)  # 以 `.` 來表示欲傳入的變數
